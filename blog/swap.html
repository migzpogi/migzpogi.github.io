linkIn-Place Swapping

Swap two values x and y in-place, without using any helper variables.

swap.py
1linkx = 5

2linky = 10

3link

4linkx ^= y 

5linky ^= x

6linkx ^= y

  • This is shorthand notation for x = x^y
  • A short review of bitwise operator (XOR):chevron_right
    1link| x | y | x^y |

    2link|:-:|:-:|:---:|

    3link| 0 | 0 |  0  |

    4link| 0 | 1 |  1  |

    5link| 1 | 0 |  1  |

    6link| 1 | 1 |  0  |

  • Values are: x = x^y -> x = 5^10 -> x = 0101^1010 -> x = 1111 -> x = 15
  • Values are: y = y^x -> y = 10^15 -> y = 1010^1111 -> y = 0101 -> y = 5
  • Values are: x = x^y -> x = 15^5 -> x = 1111^0101 -> x = 1010 -> x = 10
  • x = 10, y = 5
