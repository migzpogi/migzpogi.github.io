linkSetting up a WOTLK private server

linkSystem requirements

This guide uses a 2-Core CPU, 4 GB RAM, and 80 GB disk machine that runs on Debian 10 from Linode. I think this can serve up to 30 people max.Basic Plan - https://www.hostbarrel.net/wowprivate.php

linkUpdate sources list for clang-11

Edit sources.list by sudo vim /etc/apt/sources.list then append:

sources.list
1link$deb http://apt.llvm.org/buster/ llvm-toolchain-buster main 

2link$deb-src http://apt.llvm.org/buster/ llvm-toolchain-buster main 

3link$deb http://apt.llvm.org/buster/ llvm-toolchain-buster-10 main 

4link$deb-src http://apt.llvm.org/buster/ llvm-toolchain-buster-10 main 

5link$deb http://apt.llvm.org/buster/ llvm-toolchain-buster-11 main 

6link$deb-src http://apt.llvm.org/buster/ llvm-toolchain-buster-11 main

Install gnupg:

1link$sudo apt-get install gnupg

Add the signature:

1link$wget -O - https://apt.llvm.org/llvm-snapshot.gpg.key|sudo apt-key add -

 OK

Export:

1link$export CMAKE_C_COMPILER=clang-11

2link$export CMAKE_CXX_COMPILER=clang++-11

linkInstall AzerothCore

1link$# Go to home directory

2link$cd ~

3link$

4link$# Clone the repository

5link$git clone https://github.com/azerothcore/azerothcore-wotlk.git --branch master --single-branch azerothcore

6link$

7link$# Run cmake

8link$cd azerothcore

9link$mkdir build

10link$cd build

11link$

12link$cmake ../ -DCMAKE_INSTALL_PREFIX=$HOME/azeroth-server/ -DCMAKE_C_COMPILER=/usr/bin/clang-11 -DCMAKE_CXX_COMPILER=/usr/bin/clang++-11 -DWITH_WARNINGS=1 -DTOOLS=0 -DSCRIPTS=static

13link$

14link$# Check available cores

15link$nproc --all

16link$

17link$# Install. This will take some time.

18link$make -j <number_of_cores>

19link$make install

linkSet up MariaDB

Log in to MariaDB shell:

1link$sudo mysql_secure_installation

2link$# For the prompts: [none]/N/Y/Y/Y/Y

3link$

4link$sudo mysql

Set the database username and password:

1linkMariaDB [(none)]>GRANT ALL ON *.* TO 'adminuser'@'%' IDENTIFIED BY 'adminpass' WITH GRANT OPTION;

2linkMariaDB [(none)]>GRANT ALL ON *.* TO 'adminuser'@'localhost' IDENTIFIED BY 'adminpass' WITH GRANT OPTION;

3linkMariaDB [(none)]>Flush Privileges;

Check if users were added:

1linkMariaDB [mysql]>select user from user;

2linkMariaDB [mysql]>exit

Edit the configuration:

1link$sudo vim /etc/mysql/mariadb.conf.d/50-server.cnf

2link$

3link$# Bind Address to 0.0.0.0

4link$bind-address = 0.0.0.0

5link$

6link$# Restart.

7link$sudo /etc/init.d/mysql restart
