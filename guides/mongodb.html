linkInstalling MongoDB in Debian 9

This will install MongoDB 5.0 Community Edition using the apt package manager for 64-Bit Debian releases.

linkInitial Preparations

linkCheck if architecture is 64-bit

1link$uname -m

 x86_64

linkCreate a new user

1link$adduser mongouser

2link$usermod -aG sudo mongouser # --> -aG is for append Groups

3link$sudo su mongouser

4link$sudo whoami

 root

linkGetting MongoDB

linkImport the public key

1link$wget -qO - https://www.mongodb.org/static/pgp/server-5.0.asc | sudo apt-key add -

 OK

linkCreate sources list

1link$echo "deb http://repo.mongodb.org/apt/debian stretch/mongodb-org/5.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list

linkInstall the package

1link$sudo apt-get update

2link$sudo apt-get install -y mongodb-org

linkVerify if successful

1link$mongod --version

linkSetting up the server

linkStart the service

1link$sudo service mongod start

2link$sudo service mongod status

linkUse the mongodb shell and switch to admin database

1link$mongosh

2link$use admin

linkCreate an admin user

1link$db.createUser(

2link$  {

3link$    user: "mongoadminuser",    // --> set your username

4link$    pwd: passwordPrompt(),     // --> clear text, or prompt

5link$    roles: [

6link$      { role: "userAdminAnyDatabase", db: "admin" },

7link$      { role: "readWriteAnyDatabase", db: "admin" }

8link$    ]

9link$  }

10link$)

linkRestart and exit

1link$db.adminCommand( { shutdown: 1 } )

2link$exit

linkChange configuration

1link$sudo vim /etc/mongod.conf

Configuration file should look like this:

1linknet:

2link  bindIp: xxx.xxx.xxx.xxx  // --> IP Address

3link

4linksecurity:  // --> remove comment, and add authorization

5link    authorization: enabled

linkChange permissions

1link$sudo chown -R mongodb:mongodb /var/lib/mongodb

linkRestart the service

1link$sudo service mongod restart

2link$sudo service mongod status

linkChange socket permissions

1link$sudo chown mongodb:mongodb /tmp/mongodb-27017.sock
